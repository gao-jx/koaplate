koaplate
========

## Structure
- Typescript
- Koa2
  - Router
  - CORS
  - Logging
  - Session
  - BodyParser
  - Helmet
  - Render (ejs)
  - Static files
- Config
- Nodemon
- Webpack
  - Vue2
  - PostCSS
    - CSS Nesting
- Deploy by Shipit
- Unit Test
